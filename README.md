# activiti-basic-demo
Example code for blog post on using Activiti 7 with Spring Boot 2.x.

For more information see [Spring Boot with Activiti](http://amydegregorio.com/2019/03/21/spring-boot-with-activiti/)

To Run:
```
> gradlew bootRun
```
